# Portfolio Professionnel – BUT Informatique & École 42

Bienvenue sur le dépôt de mon portfolio numérique, réalisé dans le cadre du module **PPP (Projet Personnel et Professionnel)** à l'IUT d'Orsay (Paris-Saclay).

## Technologies utilisées

- **React 18** (Framework UI)
- **TypeScript** (Typage statique pour un code robuste)
- **Vite** (Build tool ultra-rapide)
- **CSS3** (Variables personnalisées pour le Dark/Light mode)

## Structure du projet

Le projet est entièrement décomposé en composants réutilisables (`Header`, `Skills`, `Projects`, `Experience`, `Ambitions`) pour respecter les bonnes pratiques de développement.
